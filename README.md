# Gripp — flat version (upload-proof)

All 8 files live side by side. No folders, so any upload method works.

- index.html — hub page
- maths.html / english.html / science.html / geography.html — subject pages
- gripp.css — all styling
- gripp.js — the engine (don't edit for content)
- content.js — ★ the only file you edit to change content/affiliate links

Deploy: put all 8 files at the ROOT of the repo (plus a .nojekyll file),
Settings → Pages → main branch → / (root).
